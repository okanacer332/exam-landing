# Papirus AI Landing

Papirus AI icin Cloudflare Pages uzerinde yayinlanan landing page projesi.

## Komutlar

```bash
npm install
npm run dev
npm run build
```

## Cloudflare Pages

Build command:

```bash
npm run build
```

Build output:

```text
dist
```

Production domain:

```text
https://papirus-ai.com
```
