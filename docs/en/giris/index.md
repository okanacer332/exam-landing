# Papirus AI Documentation

Papirus AI helps instructors read written exam papers faster while keeping the final academic decision under instructor control.

## Core Workflow

1. Upload an answer key or create a source-based grading frame.
2. Upload student papers from file upload or camera capture.
3. Let AI produce a draft score and feedback.
4. Review question-level evidence, adjust when needed, and export reports.

## Source Library Grading

Some departments do not work with a classic answer key. Instructors can upload course books, PDFs, notes, and other trusted material, then generate a grading frame from those sources.

The system keeps source excerpts as evidence, builds rubric-style criteria, and grades student answers against the selected source library.

## Localization

Papirus AI supports Turkish and English. Interface text, new AI feedback, invitation emails, and report headings follow the selected language. Existing historical AI outputs remain in the language they were originally generated in.
