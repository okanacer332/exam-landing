# Localization

Papirus AI supports Turkish and English.

## Landing and Documentation

The public website and documentation can be viewed in Turkish or English. English routes use the `/en` prefix.

## Console

Console language is managed from **Settings > Language**. The selected language controls:

- Sidebar and main interface labels.
- New AI feedback language.
- Invitation email language.
- Report and export headings.

## AI Output

AI transcription fields preserve the original document language. Generated feedback, rubrics, processing notes, and report labels follow the selected console language for new work.

Historical AI results are not automatically translated.
