# Source Library Grading

Source Library Grading is designed for courses where the instructor wants to grade student answers against trusted course material instead of uploading a classic answer key.

## When to Use It

- Nursing, medicine, law, social sciences, and other content-heavy programs.
- Exams where expected answers are distributed across books, PDFs, lecture notes, or official course documents.
- Assessment scenarios where the instructor still wants a reviewable rubric and point plan.

## Workflow

1. Open **Upload Sources** in the console.
2. Create a source library for the course or exam.
3. Upload PDF, TXT, or document files.
4. Process the sources so Papirus can extract readable chunks.
5. Enter the exam questions and point plan.
6. Generate a source-based answer frame.
7. Go to **Read Exam Papers**, select the generated frame, and grade student papers.

## How AI Uses Sources

Papirus sends the most relevant source excerpts to the grading model together with the visible student answers and the teacher point plan. The AI builds a source-supported checklist, awards partial credit for correct concepts, and explains the score using the source evidence.

The system does not treat the student's answer as the correct answer. It uses the uploaded course material as the grading authority and keeps instructor review in the loop.

## Best Practice

Use clear library names, upload high-quality PDFs, and write the point plan before generating the answer frame. For high-stakes exams, review the generated rubric before grading the full class.
